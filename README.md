Prerequisite
==============

* cmake
* OpenCV
* yaml-cpp


Choose python or CMake
==============


## Python

In python folder,

``` bash
python3 calib_video.py
```

## CMake

### build, Compile

``` bash
mkdir build
cd build
cmake ..
make -j4
```

### Run

In build directory

``` bash
./calib_prepro

```
